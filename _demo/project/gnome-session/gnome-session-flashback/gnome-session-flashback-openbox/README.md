## Usage

### help

run

``` sh
make
```

or run

``` sh
make help
```


### prepare

run

``` sh
make prepare
```


### build

run

``` sh
make build
```


### install

run

``` sh
make install
```


### remove

run

``` sh
make remove
```


### clean

run

``` sh
make clean
```
